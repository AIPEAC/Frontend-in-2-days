JS × HTML — Making It Alive

JavaScript manipulates the DOM (Document Object Model) — the browser's live tree of every element on the page. Change the DOM, and the page updates instantly.

01Finding Elements

// By ID — returns ONE element (or null)
const header = document.getElementById('header');

// By CSS selector — first match
const btn = document.querySelector('.btn-primary');

// All matches — returns a NodeList
const items = document.querySelectorAll('li.item');

// NodeList is array-like but NOT an Array
items.forEach(el => console.log(el));

// To use .map/.filter, convert first:
const arr = [...items];
// or: Array.from(items)
1
querySelector: Uses CSS selector syntax. #id, .class, tag, [attr], div > p — anything you'd write in CSS.
2
NodeList vs Array: querySelectorAll returns a NodeList with forEach but no map/filter. Spread converts it.
3
Live vs Static: getElementsByClassName returns a live collection. querySelectorAll returns a static snapshot. Prefer querySelector*.

02Modifying Elements

const el = document.querySelector('#demo');

// --- Text & HTML ---
el.textContent = 'Safe plain text';
el.innerHTML = '<strong>Bold</strong>';
// ⚠️ innerHTML with user input → XSS risk!

// --- Attributes ---
el.setAttribute('data-id', '42');
el.getAttribute('data-id');  // "42"
el.removeAttribute('hidden');

// --- Style (inline) ---
el.style.color = '#6c8cff';
el.style.fontSize = '1.2rem';
// CSS: font-size → JS: fontSize (camelCase)

// --- Classes (best practice) ---
el.classList.add('active');
el.classList.remove('hidden');
el.classList.toggle('dark-mode');
el.classList.contains('active'); // true
!
textContent vs innerHTML: textContent is safe — it sets plain text. innerHTML parses HTML, so user input like <script>alert('x')</script> will execute. Always sanitize.
2
classList: The modern way to manage CSS classes. toggle adds if missing, removes if present. Much cleaner than string concatenation.

03Events — Reacting to Users

const btn = document.querySelector('#myBtn');

// addEventListener (recommended)
btn.addEventListener('click', (event) => {
  console.log('Clicked!', event.target);
});

// Common events:
//   click, dblclick, mouseenter, mouseleave
//   keydown, keyup, input, change
//   submit, focus, blur, scroll, resize

// Event delegation: listen on parent
document.querySelector('ul')
  .addEventListener('click', (e) => {
    if (e.target.tagName === 'LI') {
      console.log('Item:', e.target.textContent);
    }
  });

// Remove listener
const handler = () => console.log('hi');
btn.addEventListener('click', handler);
btn.removeEventListener('click', handler);
1
event.target: The actual element clicked (could be a child). currentTarget is the element the listener is attached to.
2
Event delegation: Instead of 100 listeners, add ONE on the parent <ul>. Events "bubble" up from child to parent. Check e.target to identify which child.
3
Why not onclick=? Inline handlers mix JS into HTML, only allow one handler, and execute in global scope. addEventListener is better.

04Creating & Removing Elements

// Create new element
const card = document.createElement('div');
card.className = 'card';
card.textContent = 'New card!';

// Insert into the page
document.body.appendChild(card);

// Insert at specific position
parent.insertBefore(card, referenceNode);

// Modern: insertAdjacentHTML
container.insertAdjacentHTML(
  'beforeend',
  '<p class="note">Added!</p>'
);

// Remove
card.remove();

// Replace
parent.replaceChild(newEl, oldEl);
1
createElement + appendChild: The safe way to build DOM. No HTML string parsing, no XSS vectors.
2
insertAdjacentHTML positions: 'beforebegin' (before el), 'afterbegin' (first child), 'beforeend' (last child), 'afterend' (after el).

05Timers & Web Storage

Timers

// Run once after delay
const id = setTimeout(() => {
  console.log('Delayed!');
}, 2000); // 2 seconds

clearTimeout(id); // cancel

// Run repeatedly
const timer = setInterval(() => {
  console.log('Tick');
}, 1000);

clearInterval(timer); // stop

// requestAnimationFrame: 60fps loop
function animate() {
  // update positions...
  requestAnimationFrame(animate);
}
animate();

Web Storage

// localStorage: persists forever
localStorage.setItem('theme', 'dark');
localStorage.getItem('theme'); // "dark"
localStorage.removeItem('theme');

// Store objects: serialize first
const user = { name: "A", score: 10 };
localStorage.setItem(
  'user', JSON.stringify(user)
);
const loaded = JSON.parse(
  localStorage.getItem('user')
);

// sessionStorage: same API,
// cleared when tab closes

The Code

const btn = document.querySelector('#demo-btn');
const title = document.querySelector('#demo-title');
const list = document.querySelector('#demo-list');
let count = 0;

btn.addEventListener('click', () => {
  count++;
  title.innerText = `Clicked ${count} time(s)`;
  title.style.color =
    count % 2 ? '#e74c3c' : '#2ecc71';

  const li = document.createElement('li');
  li.innerText = `Item #${count}`;
  list.appendChild(li);
});
